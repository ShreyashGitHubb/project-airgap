# Editorial Calm UI - Design Tokens

This skill defines the visual language for the medical agent.

## Color Palette (Premium Medical)
- **Primary:** Deep Teal (#084B53) -> Deep trust and medical professionalism.
- **Secondary:** Slate Blue (#4E5D6C) -> Modern and calm.
- **Background:** Parchment White (#FAFAF9) -> approachable and clean.
- **Surface:** Glassmorphism (70% opacity white w/ 20px blur).

## Typography
- **Heading:** Serif (e.g., Georgia or Playfair Display) -> authoritative and editorial.
- **Body:** Modern Sans-Serif (e.g., Inter or Roboto) -> clean and readable.

## Components
- **Health Chips:** Use for symptom selection.
- **Reasoning Stream:** Monospaced secondary text for "Thinking" process.
- **Glass Panel:** For input and camera overlays.
- **Iconography:** Minimalist line icons (Lucide or Material symbols).

## Animation
- **Transitions:** Ease-in-out quint (300ms).
- **Thinking Loader:** Subtle pulsing dot or clinical wave.
